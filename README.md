# Math & Health Toolkit (Java, OOP)

This project is a multifunctional application developed using **Java Core**, **GUI (Swing)**, and **Utils**,  
following **Object-Oriented Programming (OOP)** principles.  

The application consists of three main modules:  
1. Basic Calculator  
2. Matrix Operations  
3. BMI Calculator  

---

## 🚀 Features

### 1. Basic Calculator
- Supports fundamental arithmetic operations: addition, subtraction, multiplication, division.
- Extra functions: **AC (clear)**, **DEL (delete)**, **% (modulus)**, **√ (square root)**.
- User-friendly GUI: dark background with green text, buttons arranged in a 5x4 grid.

### 2. Matrix Operations
- Users can adjust the number of rows and columns dynamically.
- Perform operations on two matrices:
  - **A + B** (Addition)
  - **A - B** (Subtraction)
  - **A × B** (Multiplication)
- Results are displayed in a dedicated "Result" area.
- Flexible grid input fields allow custom matrix dimensions.

### 3. BMI Calculator
- Takes user input: **Weight (kg)**, **Height (cm or m)**, and **Age**.
- Calculates Body Mass Index (BMI) and classifies the result:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Displays the BMI result in large green text.
- Provides additional information: "Status" (health condition) and "Ideal BMI range."

---

## 🛠️ Technologies
- **Java Core**
- **Java Swing (GUI)**
- **Utils** classes for modular design
- **OOP principles**:
  - **Encapsulation**: Each module is defined in separate classes.
  - **Inheritance**: Shared calculation classes are extended for specialized use.
  - **Polymorphism**: Methods are redefined for different data types and contexts.
  - **Abstraction**: Interfaces ensure modules work independently while sharing common logic.
